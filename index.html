企画概要

コードクイズ自動作成アプリ UnderCode

AIを活用してプロジェクトの構造理解からクイズ生成までを一貫支援するモダンな学習体験を目指します。

課題感

  • AI生成コードの理解が進まず構造把握が困難
  • 実践的なWebアプリに対応した学習アプリが不足
  • 高速なクイズ生成と構造化が求められる

主な機能

  • フォルダ/ファイル単位でプロジェクト読込
  • プロジェクト全体図のビジュアル表示
  • 文字起こし・LLM出力の整形とノイズ除去
  • 文脈を活かした自動補正と手動編集
  • 方針・数・文法指定によるクイズ生成
  • クイズ・QAセットのDB保存と進捗管理

技術スタック

FastAPI
Vanilla JS
paiza.io API
コンテナ実行環境

今後の機能拡張を見据え、データモデルは汎用的に設計。

想定ユーザーフロー

Draft
  1. 1

    プロジェクト読込

    コードベースをアップロードし、構造分析をトリガー。

  2. 2

    構造概要を確認

    主要フォルダ・依存関係・データフローを可視化。

  3. 3

    文脈補正と整形

    LLM出力を整形し、必要な文脈を付与してクリーンに。

  4. 4

    クイズ生成と保存

    方針・難易度・ボリュームを指定し、QAセットを生成。

  5. 5

    正答率と学習ログ管理

    回答進捗を追跡し、ユーザーの習熟度を可視化。
